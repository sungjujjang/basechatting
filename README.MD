# Base Chatting

텔레그램보다 강력한 암호화를 제공하는 채팅 서버

## Features

- Real-time messaging
- aes-256
- message history
- web, websocket api

## Installation

1. 준비중

## Spec

- fastapi
- websocket
- pyencrypt
- aes-256

## Usage

대화하고자 하는 상대방에게 키, 방 아이디 전달
전달받은 상대방은 키, 방 아이디를 제공되는 프로그램에 입력한 후 접속.
